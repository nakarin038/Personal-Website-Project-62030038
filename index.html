รายละเอียดงาน

1. ให้ผู้เรียนจัดทำเว็บไซต์ส่วนตัว
1.1 ฝากเว็บไซต์ไว้ที่บริการของ GitHub pages (https://pages.github.com/)
1.2 Source code ของเว็บไซต์ที่จัดทำต้องเก็บไว้ใน GitHub
2. เว็บไซต์จะต้องเป็นไปตามข้อกำหนดดังนี้
2.1 มีจำนวนหน้าอย่างน้อย 3 หน้า ได้แก่
• หน้าแรก (Home page)
• หน้าที่มีการใช้งาน JavaScript เช่น เกม นาฬิกา ที่เขียนด้วย JavaScript เป็นต้น (ห้ามนำงานที่ทำใน Lab มาใช้)
• หน้าข้อมูลส่วนตัว (Curriculum Vitae)
2.2 ใช้หลักการของ Progressive enhancement design
2.3 ใช้หลักการของ Responsive web design
2.4 ใช้ CSS Grid และ Flexbox

เกณฑ์การประเมินผล (คะแนนเต็ม 15 คะแนน)

1. ผลการทดสอบโดย https://web.dev/measure/ คิดเป็น 15%
โดยพิจารณาจากผลการประเมิน 3 ด้าน คือ Performance Accessibility
2. Correct HTML structure (10%)
3. Correct CSS rules (10%)
4. Progressive Enhancement (5%)
5. Best Practices/Techniques of CSS (10%)
6. Best Practices/Techniques of JavaScript (10%)
7. Good comments in your code (15%)
8. Responsive design (15%)
9. Good looking (10%)
